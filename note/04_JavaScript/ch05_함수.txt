1. 함수의 종류:
   - 익명 함수: 이름이 없어 변수에 할당하여 사용
   - 선언적 함수: 함수 이름을 직접 선언하여 정의
   - 가변 인자 함수: 매개변수 개수가 변할 수 있는 함수

2. 매개변수와 리턴값:
   - 매개변수의 개수는 유동적으로 설정 가능하며, 부족한 매개변수는 undefined로 처리
   - return을 통해 함수 실행 결과를 반환할 수 있음

3. 내부 함수와 자기 호출 함수:
   - 내부 함수는 함수 내에서 선언되는 함수로, 외부와의 충돌을 피함
   - 자기 호출 함수는 선언과 동시에 호출되며, 외부 영향 없이 독립적으로 실행

4. 콜백 함수와 클로저:
   - 콜백 함수는 다른 함수의 매개변수로 전달되어 실행
   - 클로저는 함수가 외부 변수에 접근할 수 있게 하여 함수 외부에서 접근 불가능한 변수를 유지

5. 타이머 함수:
   - setTimeout과 setInterval은 일정 시간 후 또는 반복적으로 함수 실행
   - clearTimeout과 clearInterval을 사용해 타이머 중지 가능
