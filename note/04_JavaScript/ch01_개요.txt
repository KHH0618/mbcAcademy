1. JavaScript의 역사와 활용:
   - 넷스케이프에서 Mocha라는 이름으로 시작되었고, Java와는 완전히 다른 언어.
   - 웹 문서에서 동적 기능을 제공하여 사용자 인터랙션이 가능하게 함.
   - 인터넷 발전과 함께 웹 애플리케이션으로 발전하여 풍부한 기능을 제공.

2. HTML 파일에 JavaScript 사용:
   - <script> 태그로 JavaScript 코드를 삽입할 수 있으며, 위치에 따라 실행 순서가 달라짐.
   - <head>에 넣으면 HTML <body>를 읽기 전에 실행되고, <body>에 넣으면 HTML 콘텐츠를 로드한 후 실행.

3. 오류 확인 방법:
   - 크롬의 개발자 도구(F12)를 사용하여 오류를 확인 가능.
   - 코드에서 발생한 오류의 위치와 상세 정보 확인 가능.
