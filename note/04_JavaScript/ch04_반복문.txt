1. 배열과 반복:
   - 배열은 다양한 자료형을 요소로 가질 수 있으며, length와 push 메서드를 사용해 길이를 확인하거나 요소를 추가할 수 있음.

2. while, do while 반복문:
   - while: 조건이 참일 때 계속 반복
   - do while: 조건에 관계없이 한 번은 실행하고, 조건을 확인하여 반복 여부 결정

3. for 반복문:
   - 배열 요소를 순차적으로 접근할 때 사용
   - for in: 객체의 속성이나 배열의 인덱스를 순회
   - forEach: 배열의 각 요소에 대해 콜백 함수를 호출하여 순회

4. 중첩 반복문:
   - 반복문 안에 또 다른 반복문을 사용하여 여러 차원의 데이터를 다룰 수 있음

5. break와 continue:
   - break: 반복을 중단하고 빠져나옴
   - continue: 다음 반복으로 넘어가 반복을 유지

6. 연습 예제:
   - 사용자가 입력한 구구단 출력, 특정 값 제거, 특정 조건에 맞는 반복 제어 등
