1. String 객체:
   - 문자열을 다루기 위한 객체로, 문자열 길이 확인, 부분 문자열 추출, 대소문자 변환, 특정 패턴 검색 등의 메서드를 제공
   - 예시 메서드: length, concat, substr, indexOf, lastIndexOf, toUpperCase 등

2. Date 객체:
   - 날짜와 시간을 처리하기 위한 객체로, 특정 날짜를 설정하거나 현재 날짜로부터의 차이를 계산할 수 있음
   - 날짜 간격 계산 시 getTime 메서드를 사용하여 밀리초 단위로 차이를 구할 수 있음
   - 주요 메서드: getFullYear, setDate, toDateString, toLocaleString 등

3. Array 객체:
   - 배열을 생성하고 관리하는 객체로, 배열의 길이 확인, 요소 추가 및 삭제 등의 기능 제공
   - 기본 메서드 및 속성을 통해 배열 요소를 쉽게 관리할 수 있음
