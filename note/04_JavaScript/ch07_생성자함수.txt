1. 생성자 함수 개요:
   - 생성자는 객체를 생성할 때 호출되는 함수이며, new 키워드를 통해 객체를 생성
   - 생성자 함수로 만든 객체를 인스턴스라 부르며, instanceof를 통해 특정 생성자 함수로 생성된 인스턴스인지 확인 가능

2. 프로토타입:
   - 생성자 함수로 생성된 객체들이 공통으로 사용하는 공간
   - 객체마다 중복 생성되는 메서드를 프로토타입에 정의하여 메모리 효율을 높임

3. new 키워드:
   - new 키워드를 통해 함수를 호출하면 객체를 위한 공간을 생성하고, this는 해당 객체를 참조
   - new 키워드 없이 호출하면 일반 함수로 작동하며, this는 window 객체를 참조

4. ECMAScript 6 이상의 클래스:
   - constructor 메서드를 통해 생성자를 정의하며, 클래스 내부에 메서드 정의 가능
   - 프로토타입을 활용하여 클래스 인스턴스마다 고유한 메서드를 정의 가능
