1. if 조건문:
   - 조건에 따라 실행문을 분기하며, if, if else, if else if 문 형태로 사용 가능
   - 예시: 특정 시간에 따라 오전과 오후 인사를 표시하거나 이미지 출력을 다르게 처리

2. switch 조건문:
   - 여러 조건을 비교할 때 사용하며, 각 case에 맞는 코드가 실행되고 break로 종료
   - 예시: 요일에 따라 배경색을 변경하는 기능 구현

3. 삼항 연산자:
   - 간단한 조건에 대해 조건 ? 참일 때 실행 : 거짓일 때 실행 형태로 사용
   - 조건을 짧고 간결하게 표현할 때 유용

4. 내장 객체:
   - document: 웹 페이지 콘텐츠 접근
   - Date: 현재 날짜와 시간에 접근 가능
   - Math: 수학적 계산에 사용되는 함수와 상수 포함

5. 비교 연산자:
   - ==는 값만 비교, ===는 값과 자료형 모두 비교
