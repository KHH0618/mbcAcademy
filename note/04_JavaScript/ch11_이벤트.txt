1. 이벤트 개념:
   - 사용자의 동작이나 특정 조건에 반응하는 동작으로, 클릭, 마우스 움직임, 키보드 입력 등이 해당
   - 주요 이벤트 유형: 마우스, 키보드, 입력 양식, 터치 이벤트 등

2. 이벤트 모델:
   - 인라인 이벤트: HTML 태그 내 직접 정의
   - 기본 이벤트 모델: JavaScript로 이벤트 속성에 함수 할당
   - 표준 이벤트 모델: addEventListener를 사용해 다수의 이벤트 리스너 연결 가능

3. this 키워드 사용:
   - 이벤트가 발생한 요소를 참조하는 키워드로, 해당 요소의 스타일이나 속성을 동적으로 변경 가능

4. 이벤트 강제 실행:
   - 특정 이벤트를 강제로 실행하여 연관 동작을 수행 가능

5. 디폴트 이벤트 제거:
   - 기존 HTML 요소에 내장된 기본 동작을 취소하려면 preventDefault 또는 반환값을 false로 설정
