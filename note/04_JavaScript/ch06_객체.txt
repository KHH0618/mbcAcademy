1. 객체 개요:
   - 객체는 서로 관련된 변수와 함수를 묶어서 이름을 붙인 자료형으로, 속성(property)과 메서드(method)로 구성됨
   - 배열은 여러 변수를 모아 놓은 형태이며, typeof로 확인 시 object로 나타남

2. 속성 추가와 제거:
   - 객체에 속성을 동적으로 추가하거나 delete 키워드를 통해 속성을 제거 가능

3. 객체와 반복문:
   - 객체의 속성에 접근하기 위해 for in 반복문 사용
   - 배열의 경우 일반 for문과 forEach를 사용하여 요소 접근 가능

4. 객체 생성 함수:
   - 함수를 사용하여 객체를 생성할 수 있으며, 이 함수는 속성 및 메서드를 포함하는 객체를 반환

5. 데이터 관리:
   - 객체 배열을 사용하여 데이터 관리 가능
   - 각 객체에 메서드를 추가하여 총점, 평균 등을 계산하고 출력
