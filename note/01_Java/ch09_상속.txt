1. 상속(Inheritance)이란 부모클래스가 가지고 있는 속성과 메서드를 활용할 수 있는 개념이다.
2. 상속이란 기존의 클래스(super)를 재사용해서 새로운 클래스를 작성하는 것이다.
3. 상속을 받게된 자식클래스의 경우 부모클래스의 모든 인스턴스를 받게되며 오버라이딩또한 가능하다.
4. 모든 클래스는 object클래스로 부터 상속을 받는다 따라서 Object -> S class -> A&B&C class순으로 상속이 된다.
5. 상속이 없는 클래스는 자동적으로 Object클래스를 상속받게된다.
6. 오버라이드란 부모클래스의 method를 자식클래스에서 재정의 하는것이다. 부모클래스의 메소드를 자식클래스에서 동일한 이름으로 재정의 하게되면 해당 메소드는 부모클래스가 아닌 자식클래스의 메소드가 호출되게 된다.
7. 생성자 오버로딩이란 인자의 타입이 다르면 같은 이름의 메소드라도 다른 기능으로 중복정의가 가능하다.
8. 상속을 받기 위해선 
접근제어자 [final/abstract] class 클래스이름 extends 상위클래스(super class) {
	추가할 멤버변수선언;
	생성자;
	추가할 메소드선언;
    }
위와 같이 작성하면 된다.
9. 상속은 다른사람이 만들어놓은 유용한 메소드의 집합이 있다면 그것을 활용하여 빠르게 결과물을 만들 수 있다.
10. 상속을 활용하면 다양한 객체를 상속을 통해서 하나의 객체로 묶을수 있다.
