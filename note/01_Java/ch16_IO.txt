1. 입출력이란 프로그램에게 입력하는것을 입력(input)이고 프로그램에서 어떤것을 출력하는것을 출력(output)이라한다
2. 자바는 파일을 읽거나 쓰기위해 스트림을 통해 파일을 열어야 하면 스트림을 닫지 않으면 다른 프로그램은 해당파일에 접근 할 수 없다.
3. InputStream : 1byte단위 입력 API. 이미지, 동영상 등의 데이터에 주로 사용
4. OutputStream : 1byte단위 출력 API. 이미지, 동영상 등의 데이터에 주로 사용
5. Reader : 2byte단위 입력 API. 문자열에 주로 사용
6. Writer : 2byte단위 출력 API. 문자열에 주로 사용
7. 입출력작업은 항상 close를 해야 다른 프로그램이 파일에 접속할 수 있고 어떤 method는 close를 하지않으면 저장이 안되기에 반드시 close를 해야한다
8. IOException은 스트림객체를 생성할때 반드시 예외처리를 해야하며 finally구문에는 보통 close선언을 한다.
