1. 자바란? 가상머신으로 구동되는 프로그램 언어이며 운영체제의 영향을 받지 않는다. 매모리 관리또한 가상머신이 관리해준다. 객체지향 언어로 유지보수를 효율적으로 진행할 수 있다.
2. 자바는 윈도우기준 메모장과 CMD로 구동할수있지만 효율적인 코딩을 위해 자바 통합계발환경(IDE)프로그램중 하나인 이클립스를 이용한다.
3. 자바의 데이터 타입은 기본자료형과 참조자료형이 있다. 참조자료형은 변수에 값을 저장하는것이 아닌 값의 메모리주소를 저장후 메모리주소를 참조하여 값을 불러온다.
4. 자바의 변수는 데이터타입 변수명 으로 선언하며 같은 데이터 타입일 경우 복수 선언이 가능하다.
5. 자바의 데이터타입 형변환은 명시적 형변환과 묵시적 형변환이 있는데 큰 데이터 타입에 작은 데이터 타입이 들어갈 경우 묵시적, 작은 데이터 타입에 큰 데이터 타입이 들어갈경우 명시적으로 형변환을 하며 작은 데이터 타입에 큰 데이터 타입이 들어갈경우 데이터 손실이 일어날 수 있다.
