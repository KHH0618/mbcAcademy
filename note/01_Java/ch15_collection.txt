1.Collection이란 자료구조이다. 배열과 비슷한 구조에 다수의 데이터를 쉽게 처리할 수 있는 method를 지원한다.
2.자바에서 제공되는 콜렉션 자료구조들 : 1) List(순서, 중복허용), 2) Set (순서X,중복X), 3)Map(Key,Value)
3.List : 자료구조 중 가장 많이 사용하고 쉽게 사용할 수 있다. ArrayList - 배열과 매우 비슷하고 인덱스가 존재하며 데이터는 중복을 허용한다.
4.Map : 자료구조는 인덱스 대신 키 값으로 데이터를 엑세스 한다. 다만 키값으로 데이터를 엑세스 해야함으로 키값은 유니크 해야한다
5.Set : 자료구조는 데이터의 순서가 없고 중복된 데이터는 허락하지 않는다. 데이터에 접근하기 위해선 Iterator(반복자)를 이용해야한다.
6.Iterator(반복자) : 반복자는 set자료구조에서 자주 사용하며 hashnext를 이용하여 다음 데이터가 있는지 확인하고 next를 통해 가져올 수 있다.
