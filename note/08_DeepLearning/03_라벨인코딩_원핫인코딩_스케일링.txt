1. 라벨 인코딩(Label Encoding):
   - 범주형 데이터를 수치형 데이터로 변환하여 모델이 이를 처리할 수 있도록 합니다.
   - 예를 들어, '사과', '바나나', '체리'라는 범주형 데이터를 각각 0, 1, 2로 변환합니다.
   - 이러한 변환은 모델이 범주형 데이터를 이해하고 학습하는 데 도움을 줍니다.

2. 원핫 인코딩:
   - 범주의 순서 관계가 제거되며 각 범주가 독립적인 열로 표현됨.
   - 예를 들어, 데이터가 ['사과', '바나나', '체리']인 경우:
	(to_categorical)
	'사과' → [1, 0, 0]
	'바나나' → [0, 1, 0]
	'체리' → [0, 0, 1]
	(pd.get_dummies)
	'사과' → [0, 1, 0, 0]
	'바나나' → [0, 0, 1, 0]
	'체리' → [0, 0, 0, 1]

3. 스케일링(Scaling):
   - 데이터의 범위를 조정하여 모델 학습의 효율성을 높입니다.
   - 주요 스케일링 기법으로는 다음이 있습니다:
     - 표준화(Standardization): 데이터를 평균이 0, 표준편차가 1이 되도록 변환합니다.
     - 정규화(Normalization): 데이터를 0과 1 사이의 값으로 변환합니다.
   - 이러한 스케일링은 모델이 다양한 범위의 데이터를 효과적으로 학습할 수 있도록 돕습니다.
