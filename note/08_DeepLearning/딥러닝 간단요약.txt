1. 딥러닝 프레임워크:  
   - TensorFlow와 Keras는 딥러닝 모델 설계와 학습을 돕는 주요 라이브러리로, Google Colab 환경에서 사용 가능하다.  
   - 설치 명령어는 pip install tensorflow==2.10.  

2. 지도 학습(Supervised Learning):  
   - KNN과 같은 알고리즘을 통해 레이블이 있는 데이터를 학습하며, 회귀 및 분류 문제에 사용된다.  
   - 데이터와 모델 간의 오차를 줄이기 위해 경사하강법과 손실 함수(MSE, MAE 등)를 활용한다.  

3. 인공신경망(ANN):  
   - 인간 뉴런을 모사한 인공 뉴런은 가중치가 있는 입력 신호를 처리하며, 활성화 함수를 통해 출력값을 결정한다.  
   - 다층 퍼셉트론(MLP)은 XOR 문제 등 비선형 문제를 해결하는 딥러닝의 기본 구조다.  

4. 컨볼루션 신경망(CNN):  
   - 이미지 데이터를 처리하기 위한 주요 구조로, 특징을 추출하는 컨볼루션 레이어와 정보를 축소하는 맥스풀링 레이어로 구성된다.  
   - 플래튼 레이어를 사용해 다차원 데이터를 1차원으로 변환해 분류 작업에 활용한다.
