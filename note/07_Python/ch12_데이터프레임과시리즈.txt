1. 판다스(Pandas) 개요:  
   - 1차원 데이터 구조인 시리즈(Series)와 2차원 데이터 구조인 데이터프레임(DataFrame)을 제공하며, 데이터 분석과 전처리에 널리 사용된다.  
   - 데이터프레임은 행과 열로 구성된 엑셀과 유사한 구조를 가지며, 시리즈는 데이터프레임의 열(Column) 하나에 해당한다.  
   - 파일 입출력, 데이터 추가/삭제, 데이터 병합, 정렬, 통계분석 등 다양한 기능을 제공한다.

2. 데이터프레임 생성:  
   - 딕셔너리나 리스트, CSV 파일, Excel 파일 등을 이용하여 생성할 수 있다.  
   - scikit-learn, seaborn 등에서 제공하는 데이터셋을 쉽게 데이터프레임으로 변환 가능하다.

3. 데이터 조회와 수정:  
   - 단일 열 조회: 열 이름이나 loc, iloc, 조건을 이용해 데이터 조회  
   - 데이터 삭제: 행 또는 열을 삭제 가능하며, inplace=True나 할당으로 원본 데이터를 수정 가능  
   - 데이터 추가: 새로운 열을 추가하거나, 시리즈, 딕셔너리, 리스트 등을 사용해 행 추가 가능  

4. 데이터프레임 병합과 연결:  
   - merge()를 이용한 데이터프레임 병합과 concat()을 이용한 연결 지원  
   - 병합 시 how, left_on, right_on 등의 매개변수를 사용해 조인 방식과 기준을 설정 가능  

5. 정렬:  
   - 행 이름, 열 이름, 값 기준으로 정렬 가능  
   - sort_index(), sort_values() 함수 제공  

6. 기초 통계 분석:  
   - min(), max(), mean(), median() 등의 함수로 기본 통계 분석  
   - 요약 통계량 제공 (count, unique, top, freq 등)  
   - 분산, 표준편차, 공분산, 상관계수 등 고급 통계 분석 지원  

7. 데이터 그룹화 및 집계:  
   - groupby()를 통해 데이터를 그룹화하여 통계 함수 적용 가능  
   - 단일 열 또는 다중 열 기준으로 그룹화 가능  

8. 데이터 구조 변경:  
   - melt()를 이용한 언피벗, pivot_table()을 이용한 피벗팅 기능 제공  
   - 와이드 포맷과 롱 포맷 간의 변환 지원  

9. 데이터프레임에 함수 적용:  
   - apply(), applymap(), map() 함수로 사용자 정의 함수 적용 가능  
   - 각 행, 열 또는 요소별로 연산 수행  

10. 파일 입출력:  
    - CSV, Excel, JSON, HTML 등 다양한 파일 포맷을 읽고 저장할 수 있는 기능 제공
