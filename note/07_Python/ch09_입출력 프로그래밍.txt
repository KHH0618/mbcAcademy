1. 파일에 데이터 저장하고 불러오기:  
   - open() 함수를 사용해 파일을 열고 작업하며, 모드(r, w, a, b)를 지정해 사용한다.  
   - 파일 읽기: readline(), readlines() 메서드를 사용하며, 데이터를 처리할 때 strip()으로 공백 제거.  
   - 파일 쓰기: write() 함수로 데이터를 파일에 작성.  

2. 피클을 이용한 객체 저장하고 불러오기:  
   - 피클링은 객체를 바이트 스트림으로 변환해 저장하는 방법이다.  
   - 파일을 열 때 바이너리 모드(wb, rb)를 사용해야 하며, dump()와 load()로 데이터를 직렬화 및 역직렬화한다.  
   - 피클링된 데이터는 다시 불러와 원래 객체 형태로 복원할 수 있다.  

3. CSV 형식 파일 읽기/쓰기:  
   - csv 모듈을 사용해 파일 작업 수행.  
   - reader()와 writer() 함수로 데이터를 읽거나 쓰며, DictReader와 DictWriter로 딕셔너리 데이터를 처리 가능.  
   - delimiter와 quotechar 속성을 설정해 데이터 형식을 맞출 수 있다.  

4. JSON 데이터 저장하고 불러오기:  
   - json 모듈의 dump()와 load()를 사용해 JSON 형식 데이터 저장 및 불러오기 가능.  
   - JSONEncoder를 이용해 객체를 JSON 형식으로 변환하고, object_hook으로 데이터를 객체화할 수 있다.  

5. HDF5 파일 읽기/쓰기:  
   - 대용량 데이터를 저장하고 처리하기 위한 형식으로, 판다스의 read_hdf()와 to_hdf() 또는 h5py 모듈의 File 클래스를 사용한다.  
   - 데이터 구조를 계층적으로 저장하며, HDFView를 통해 내용을 시각적으로 확인 가능.
