1. 조건문:  
   if, if ~ else, if ~ elif ~ else 구문을 통해 조건에 따라 코드 실행을 분기한다. 조건식이 참이면 해당 블록을 실행하며, else 또는 elif를 통해 추가적인 조건을 처리할 수 있다.

2. 반복문:  
   for문은 리스트, 튜플, 딕셔너리 등 나열 가능한 자료를 순회하며, range()를 사용해 인덱스 기반 반복도 가능하다. while문은 조건이 참일 동안 반복하며, 조건이 거짓이 되면 종료한다. 반복문 내에서 break와 continue로 흐름을 제어할 수 있다.

3. 중첩 루프:  
   for문이나 while문을 중첩해 사용할 수 있으며, 2차원 리스트나 3차원 리스트의 데이터를 처리할 때 유용하다. 반복문 중첩 시 각 루프의 들여쓰기 간격을 동일하게 유지해야 한다.

4. 중첩 루프 탈출:  
   중첩된 반복문을 탈출하기 위해 break와 continue를 사용하며, 플래그나 예외처리를 통해 여러 개의 반복문을 한꺼번에 빠져나올 수도 있다.
