1. 리스트:  
   리스트는 여러 개의 값을 저장하는 데이터 구조로, 대괄호를 사용해 만들며 인덱스를 통해 접근하고 수정할 수 있다. 슬라이싱과 append, extend, insert 등의 메서드를 통해 데이터를 추가할 수 있다.

2. 튜플:  
   튜플은 소괄호를 사용해 만들며, 리스트와 달리 데이터를 수정할 수 없는 읽기 전용 구조이다. 속도가 빨라 수정이 필요 없는 데이터를 저장하는 데 적합하다.

3. 딕셔너리:  
   딕셔너리는 키와 값의 쌍으로 구성된 자료 구조로, 중괄호를 사용해 만든다. 키는 유일해야 하며 값을 중복할 수 있다. 키를 통해 데이터를 접근하며, items, keys, values 메서드를 통해 딕셔너리의 다양한 정보를 얻을 수 있다.

4. 셋:  
   셋은 순서가 없고 중복을 허용하지 않는 데이터 구조로, 중괄호를 사용해 만든다. 중복 없이 데이터 집합을 관리하는 데 사용된다.

5. enumerate:  
   enumerate는 리스트나 튜플 등의 반복 가능한 객체에서 인덱스와 값을 동시에 반환하는 함수로, 루프에서 인덱스가 필요한 경우 유용하다.
