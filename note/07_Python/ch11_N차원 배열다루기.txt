1. Numpy 소개:  
   Numpy는 과학 컴퓨팅에 사용되는 파이썬의 핵심 패키지로, N차원 배열 객체와 선형대수, 푸리에 변환, 난수 생성 등의 기능을 제공한다. 주요 배열 객체는 동일한 데이터 타입의 다차원 배열로 구성된다.

2. Numpy 배열 생성:  
   배열 생성 방법으로 array, arange, zeros, ones, empty 등의 함수가 사용된다. 초기값 배열이나 연속된 값을 가지는 배열 생성도 가능하다.

3. 배열 조작:  
   배열의 모양 변경을 위해 reshape, ravel, T(전치) 등이 사용된다. 배열을 합칠 때는 hstack, vstack, dstack을, 배열 분할에는 hsplit, vsplit, dsplit을 활용할 수 있다.

4. 고급 인덱싱:  
   기본 인덱싱, 슬라이싱 외에도 다차원 인덱싱을 지원한다. 논리 배열이나 ix_()를 활용한 복잡한 인덱싱도 가능하다.

5. 복사와 뷰:  
   얕은 복사(할당)는 동일 데이터를 공유하고, 깊은 복사(copy)는 독립적인 데이터를 생성한다. 뷰는 얕은복사와 유사하게 데이터는 공유하지만 배열의 속성을 따로 가진다

6. 선형대수 연산:  
   행렬 곱(dot, @), 역행렬(inv), 대각합(trace), 연립방정식 풀이(solve) 등을 지원하며, 최소제곱법(Least Square Method)도 적용할 수 있다.
