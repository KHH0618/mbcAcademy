1. JDBC 개요:  
   - JDBC는 자바 프로그램에서 SQL문을 실행하여 데이터베이스와 상호작용할 수 있도록 하는 API로, 다양한 데이터베이스를 하나의 프로그램으로 관리할 수 있게 한다. 오라클과의 연결을 위해 OracleDriver를 사용하며, 이클립스와 연동하여 사용한다.

2. JDBC 사용 순서:  
   - JDBC 드라이버를 로드하고 데이터베이스와 연결한 뒤, SQL문을 실행하여 결과를 처리한다. 데이터베이스 작업이 끝나면 연결을 해제하여 자원을 관리한다.

3. Statement 객체와 ResultSet:  
   - Statement 객체는 SQL문을 실행하며, executeQuery()는 SELECT와 같은 결과 집합을 반환하는 SQL문에 사용하고, executeUpdate()는 테이블을 수정하는 SQL문에 사용한다. ResultSet은 쿼리 결과를 받아오며, next(), previous() 등을 통해 결과를 순회할 수 있다.

4. 데이터베이스 연동 JSP 예제:  
   - JSP 페이지에서 JDBC를 사용하여 데이터베이스에 연결하고 데이터를 조회하는 예제를 작성한다. 부서번호로 사원을 조회하거나 사원명을 검색하는 기능 등을 통해 동적 데이터를 웹 페이지에 출력할 수 있다.
