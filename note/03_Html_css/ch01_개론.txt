1. 웹(Web):
웹은 전 세계의 컴퓨터를 연결해 정보를 공유하는 인터넷 서비스의 일종이다.
다양한 네트워크 프로토콜을 사용하며, 대표적으로 HTTP, FTP, DNS, IP 등을 포함한다.

2. HTML:
HyperText Markup Language의 약자로, 웹 문서를 작성하기 위한 언어이다.
HTML은 텍스트뿐 아니라 이미지와 링크 등의 요소를 표시하며, 태그(tag)로 구조화한다.
HTML5는 멀티미디어 기능을 제공하고 CSS3와의 호환성을 강화한다.

3. CSS:
Cascading Style Sheets의 약자로, HTML 문서의 디자인을 담당한다.
웹 페이지의 레이아웃, 색상, 폰트 등의 스타일을 정의해 사용자 경험을 개선한다.

4. JavaScript와 jQuery:
JavaScript는 웹 문서에 동적인 기능을 추가하며, jQuery는 이를 쉽게 사용할 수 있도록 돕는 라이브러리이다.
AJAX 기술은 JavaScript를 활용해 특정 부분을 업데이트하거나 서버와 데이터를 교환한다.

5. 웹 표준:
웹 표준을 준수하면 다양한 브라우저와 기기에서 일관성 있게 웹 페이지를 표시할 수 있다.
HTML, CSS, JavaScript의 역할을 분리하여 유지 보수와 접근성을 높인다.

6. 개발 도구:
Visual Studio, EditPlus, Notepad++, 이클립스 등의 편집기와 크롬 브라우저 설치 방법도 포함한다.
