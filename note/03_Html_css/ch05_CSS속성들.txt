1. CSS 단위:
   - 자주 사용하는 단위: em, %, px, pt.
   - url(): 배경 이미지나 폰트 경로 지정.

2. 디스플레이 관련 속성:
   - display: 요소를 block, inline, inline-block, none으로 표시.
   - visibility: hidden은 공간을 유지하며 숨기고, display: none은 공간까지 제거.
   - opacity: 투명도 설정.

3. 레이아웃과 박스 모델:
   - margin과 padding: 요소의 바깥과 안쪽 여백 설정.
   - box-sizing: 전체 박스 크기를 설정할 때 테두리를 포함할지 여부.
   - border: 테두리 두께, 스타일, 색상.

4. 텍스트 관련 속성:
   - font-family, font-size, font-weight, font-style, line-height, text-align, text-decoration.
   - 웹폰트, 로컬 폰트 지정 가능.

5. 포지션과 플로트:
   - position: 요소 위치 설정 (absolute, fixed, relative, static).
   - float: 요소를 왼쪽 또는 오른쪽에 배치하여 레이아웃 설정.
