1. 기본 선택자:
   - *: 전체 선택자 (페이지 초기화).
   - 태그, ID(#), 클래스(.) 선택자.

2. 혼합 및 속성 선택자:
   - 혼합 선택자 (태그, ID, 클래스 조합).
   - 속성 선택자 [속성=값]: 특정 속성을 가진 요소 선택.

3. 후손, 자손 선택자:
   - 후손 선택자 (공간 내 모든 해당 요소).
   - 자손 선택자 (>) (바로 아래 요소만 선택).

4. 동위(형제) 선택자:
   - +: 바로 다음 형제 요소.
   - ~: 모든 형제 요소 선택.

5. 반응 및 상태 선택자:
   - :hover: 마우스를 올렸을 때 반응.
   - :focus, :disabled, :enabled: 특정 상태에서 적용.

6. 구조 선택자:
   - :nth-child(), :first-child, :last-child: 요소 구조에 따른 선택.

7. 링크 및 문자 선택자**:
   - 링크 선택자 (a[href]), 문자 선택자.

8. 부정 선택자:
   - :not(): 특정 요소를 제외하고 적용.
