1. HTML 구조:
   - <!DOCTYPE html>: HTML5 문서임을 명시.
   - <html lang="ko">: 언어 설정 (검색 엔진이 인식).
   - <head>, <body>: 문서 정보와 콘텐츠를 담는 부분.

2. 문자 태그:
   - 제목 태그 (`<h1>` ~ `<h6>`), 본문 (`<p>`), 줄바꿈 (`<br>`), 강조 (`<b>`, `<i>`), 링크 (`<a>`).
   - `<sup>`, `<del>` 등의 특수 태그 사용 가능.

3. 목록 태그:
   - `<ul>`: 순서 없는 목록 (원, 사각형 등).
   - `<ol>`: 순서 있는 목록 (숫자, 문자 등).

4. 표 태그:
   - `<table>`, `<tr>`, `<td>`로 구성. 제목 셀 (`<th>`), 머리/본문/하단 구분 가능 (`<thead>`, `<tbody>`, `<tfoot>`).

5. 이미지 태그:
   - `<img src="경로" alt="설명" width="너비" height="높이">`로 이미지 삽입.

6. 오디오 태그:
   - `<audio src="경로" controls autoplay loop>`로 오디오 재생 가능 (HTML5 지원).
